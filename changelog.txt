- Added new "Labyrinth" secret seed
- Added new Manual Ecotone Mapping functionality for world generation
	There's a new button right next to the name plate for worlds which enables/disables this functionality
- Added new Ecotone Debugging visualization for internal and cross-mod use
	Enable this in the config if you want to see how the ecotone mapper maps out ecotones!

- Fixed Oasis crash issue with Remnants
- Fixed naturally generated Lapis Dressers not generating with chests properly
- Added small loot pool for Lapis Dressers
- Allowed Candelabras and Candles to spawn on Lapis Dressers
- Fixed Gilded Greatbow doing nonsense in multiplayer
- Dunce Crabs now fall substantially slower in water
- Fixed duplicate application of matrix to wavy sign text, causing visual issues
- Fixed the ecotone mapper considering the ocean as a desert by force
- Fixed the ecotone mapper mislabeling corrupt ecotone entries (applying to the next entry instead of the current)
- Fixed rightmost ecotone entry not having ocean as its right bound
- Added proper ecotone entries for both oceans
- Fixed issue where certain corrupt/crimson patches would spam switching ecotone entries instead of flowing continuously
- Probably fixed some additional issues with the ecotone mapper I forgot about
- Added wind sway to all of our custom lanterns
- Fixed our custom chandeliers not having the exact same sway visuals as vanilla's chandeliers
- Fixed Otherworldly Radio not being consumable despite being placeable, allowing it to be infinitely farmed
- Fixed Savanna Grass Seeds placing multiple times per use
